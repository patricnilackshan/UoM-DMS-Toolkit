# Contributing to UoM DMS Toolkit

Thanks for taking the time to help improve this project! Here are a few ways you can contribute.

---

## 🐛 Report a Bug

Open an [issue](../../issues/new) and include:

- Which method you used — Google Colab or GitHub Actions
- The type of link you tried (YouTube / torrent / direct / M3U8) — not the actual link
- What happened vs what you expected
- A screenshot or copy of the error message if possible

---

## 💡 Suggest a Feature

Open an [issue](../../issues/new) and describe:

- What you want the toolkit to do
- Why it would be useful for UoM students

---

## 🔧 Submit a Fix or Improvement

1. Import this repo as a private copy or fork it
2. Make your changes
3. Open a [pull request](../../pulls) with a short description of what you changed and why

---

## 📌 Notes

- Keep changes focused — one fix or feature per pull request
- Test your changes in Google Colab before submitting
- This toolkit is for UoM students — keep that use case in mind
